# Changelog

## Version 1.0.0 (2024-12-20)

### Initial Release
- ✨ Schedule shutdown, restart, or logout with custom timer (hours, minutes, seconds)
- ✨ Execute actions immediately without timer
- ✨ Clean, native macOS interface with SwiftUI
- ✨ Beautiful gradient app icon with power symbol
- ✨ Sound notification before action execution
- ✨ Input validation (numbers only)
- ✨ Standard macOS menu bar with About dialog
- ✨ Proper app bundle with icon integration
- ✨ Timer cancellation support
- ✨ Real-time countdown display

### Technical Details
- Built with Swift and SwiftUI
- Requires macOS 13.0 or later
- Uses AppleScript for system control
- Native macOS look and feel
